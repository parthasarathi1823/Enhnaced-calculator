Enhanced Calculator (Python)
A command-line based enhanced calculator built using Python.
This project demonstrates the use of decorators, closures, and basic error handling.

Features:
Basic arithmetic operations (add, subtract, multiply, divide, power)
Logs function execution using decorators
Stores the last calculated result using closures
Option to view and clear the last result
Handles invalid input and runtime errors

Concepts Used:
1.Functions
<br>
2.Decorators
<br>
3.Closures
<br>
4.Exception handling
<br>
5.Dictionaries

How to Run
1.Clone the repository
<br>
2.Run the Python file:
